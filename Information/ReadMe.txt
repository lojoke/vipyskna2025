-- Відомості про файл запуску
Для запуску проекту необхідно мати встановлений PHP на вашому комп’ютері. Виконайте наступні кроки:

Відкрийте термінал у Visual Studio Code або іншому терміналі.
Перейдіть до директорії проекту, використовуючи команду cd, наприклад: "cd шлях/до/проекту"

Запустіть вбудований сервер PHP за допомогою команди: "php -S localhost:8000"

Відкрийте веб-браузер та перейдіть за адресою: "http://localhost:8000/main.html"

-- Відомості про особливості установки та роботи програми
Клонування проекту: Завантажте всі файли проекту до вашого комп’ютера або клонувати репозиторій з GitHub (якщо доступно).

Встановлення PHP:
Завантажте та встановіть PHP з офіційного сайту: php.net.
Додайте PHP до системної змінної PATH, щоб мати можливість запускати команди з будь-якої директорії.

Запуск серверу: (Описаний вище у категорії "Відомості про файл запуску")

Навігація по веб-ресурсу: Використовуйте навігаційне меню для переходу між розділами "Introduction", "States", "History" та "Conclusion".
Натискайте на кнопки та інтерактивні елементи для отримання додаткової інформації.
Перемикання теми:

Використовуйте кнопку перемикання теми у навігаційному меню, щоб змінити вигляд сторінки між світлим та темним режимами.
Вибір теми зберігається у вашому браузері, тому при наступному відкритті сторінки обрана тема залишиться активною.
Взаємодія з інтерактивною картою:

Наведіть курсор на будь-який штат на карті, щоб побачити його назву та коротку інформацію.
Клацніть на штат для перегляду детальної інформації у вигляді спливаючих вікон чи підказок.
